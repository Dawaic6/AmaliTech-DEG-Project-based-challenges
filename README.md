simple React-based visual chatbot flow builder.
Create, edit, and test conversation flows using an interactive UI.

🚀 Features
Visual flow builder (drag & connect nodes)
Chatbot simulation
Undo / Redo support
Local storage saving
🛠️ Installation
npm install
npm run dev

App runs at:

http://localhost:5173/
📦 Build
npm run build
🌐 Deployment

You can deploy easily using:

Netlify
